## 📌 Описание

В этом проекте представлены примеры и задания, связанные с коллекциями **Map** и **Set** в JavaScript.  

Цель репозитория — помочь понять, как работают эти структуры данных, как они используются и чем отличаются от обычных объектов или массивов.

Структуры Map и Set позволяют:

- хранить пары ключ‑значение с ключами любых типов;
- хранить уникальные значения без дубликатов;
- эффективно выполнять операции поиска, удаления и вставки элементов.
